# WebGL-PSU

## Capabilities

1. Landscape
2. Skybox
3. Camera
4. Objects
5. Object rotation
6. Relief texturing

## Demo

https://mishhan-webgl-psu.herokuapp.com/

## Installation

* `git clone <repository-url>` this repository
* `cd WebGL-PSU`
* `npm install`

## Local

* `npm run start-app`
* Visit app at [http://localhost:5500](http://localhost:5500)
